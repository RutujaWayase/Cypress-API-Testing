# Cypress-API-Testing

Open Cypress in headless mode: 'npx cypress run'
Open Cypress in headed mode: 'npx cypress open'
 
EX: to run single file 'npx cypress run --spec cypress\e2e\APITesting\HTTPRequests.cy.js'

OAuth 2.0 Authentication.
